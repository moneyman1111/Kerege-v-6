## Что нужно обновить

### Контактная информация:
В `index.html` найдите и замените:

```html
<!-- Instagram -->
<a href="https://instagram.com/kerege.synak" target="_blank">
<!-- Замените на реальную ссылку -->

<!-- WhatsApp -->
<a href="https://wa.me/996555123456" target="_blank">
<!-- Замените на реальный номер -->

<!-- Телефон -->
<a href="tel:+996555123456" class="btn btn-primary btn-call">
<!-- Замените на реальный номер -->

<p class="contact-info">+996 555 123 456</p>
<!-- Замените на реальный номер -->

<p class="contact-address">г. Бишкек, ул. Примерная, 123</p>
<!-- Замените на реальный адрес -->
```

### Фотографии команды (опционально):
Сейчас используются emoji (👨‍🏫, 👩‍🏫, 👨‍💼, 👩‍💻).

Чтобы добавить реальные фото:
1. Замените `<div class="team-avatar">👨‍🏫</div>` на `<img src="путь/к/фото.jpg" alt="Имя">`
2. Добавьте CSS для изображений:
```css
.team-photo img {
    width: 120px;
    height: 120px;
    border-radius: 50%;
    object-fit: cover;
}
```
